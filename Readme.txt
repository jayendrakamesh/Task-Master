How to run this electron app:
Open terminal then follow the below steps

Step 1: Run "mkdir Task Master"
Step 2: Run "cd Task Master"
Step 3: Run "npm init"
Step 4: Run "npm install electron"
Step 5: Copy the source files to this directory
Step 6: Run "npm install eletcron-builder"
Step 7: Run "npm install sqlite3"
Step 8: Run "npm start"


How to build this app:

Run "npm build".
